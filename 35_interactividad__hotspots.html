3.5. Interactividad ( Hotspots)

¿Qué es un hotspot?

Si las imágenes panorámicas de 360° son el esqueleto de nuestro tour virtual, los Hotspots (puntos de interés) son el corazón y los músculos.

Un Hotspot es un marcador visual o "botón flotante" interactivo que podemos arrastrar y colocar en cualquier coordenada espacial de nuestra fotografía de 360°. Su función principal es servir de puente entre el usuario y el entorno, permitiéndole realizar acciones al hacer clic o pasar el cursor por encima de él.

Sin los hotspots, un tour virtual sería simplemente una galería de fotos fijas para mirar. Gracias a ellos, podemos transformar una imagen plana en una experiencia dinámica donde el visitante puede explorar a su propio ritmo, investigar objetos, leer descripciones, escuchar explicaciones o desplazarse de una habitación a otra como si caminara físicamente por el edificio.

🛠️ ¿Cómo se añade un Hotspot en Lapentor?

El proceso en el editor es sumamente intuitivo:

  1. Localiza la barra de herramientas interactiva en el menú lateral derecho de la pantalla.
  2. Haz clic sobre el icono del tipo de hotspot que necesites y, sin soltar el botón, arrástralo hacia el punto exacto de la imagen panorámica donde quieras situarlo.
  3. Una vez soltado sobre el lienzo, se abrirá un pequeño menú de configuración para personalizar su icono, su color, su título y el contenido multimedia que va a desplegar.

Point Hotspot

El Point Hotspot es el "portal" que conecta una habitación con otra (por ejemplo, del porche al vestíbulo). Al arrastrarlo al lienzo, se configura con estas opciones clave:

🔗 1. Destino (Sección CONTENT)

  • Select scene: Elige qué imagen 360° se abrirá al hacer clic.
  • Recuadro de vista inicial: Arrastra la miniatura para definir exactamente hacia dónde mirará el usuario nada más entrar a la nueva habitación.

🎨 2. Aspecto (Sección HOTSPOT)

  • Label: El texto flotante que verá el usuario.
  • Icon: Cambia el diseño de la flecha o botón.
  • Size (px): Regula el tamaño del icono en pantalla.
  • Password: Permite poner una contraseña para bloquear el acceso (ideal para juegos tipo Escape Room).

Image Hotspot

El Image Hotspot permite desplegar una fotografía o infografía flotante sobre el entorno 360° al hacer clic en él. Sus opciones principales son:

📁 1. Contenido a Mostrar (Sección CONTENT)

  • Choose/upload image: Botón para seleccionar o subir desde tu ordenador la imagen que se abrirá en la ventana flotante.
  • Select theme: Permite elegir el estilo visual o el marco que rodeará a la foto emergente.
  • Caption (Pie de foto): Espacio para escribir una breve descripción o título explicativo debajo de la imagen.
  • Url (Enlace web): Si pegas aquí una dirección web, el usuario será redirigido a esa página externa si hace clic directamente sobre la foto desplegada.


🎨 2. Aspecto del Marcador (Sección HOTSPOT)

  • Label: El texto indicador que flotará sobre el botón en el tour (ej: "Ver plano del aula" o "Detalle del hardware").
  • Icon: Permite cambiar el icono del marcador (por defecto viene con forma de cámara o paisaje).
  • Size (px): Modifica el tamaño del botón flotante en la escena.
  • Password: Permite bloquear la visualización de la imagen con una contraseña.

Article Hotspot

El Article Hotspot abre una ventana emergente tipo artículo de prensa o enciclopedia. Es ideal para describir la historia del centro, explicar las normas de un aula o detallar las características de un cuadro o máquina.

Sus opciones principales son:

✍️ 1. Redacción del Contenido (Sección CONTENT)

Editor de Texto Enriquecido: Funciona exactamente igual que un procesador de textos (Word o el propio eXeLearning). Te permite:

  • Aplicar formatos básicos: Negrita, cursiva, subrayado, fuentes y tamaños de letra.
  • Organizar la información: Crear listas con viñetas, alinear textos y añadir tablas.
  • Insertar elementos multimedia: Añadir hipervínculos externos (🔗), imágenes estáticas o reproductores de vídeo integrados directamente en el cuerpo del texto.
  • Código (</>): Permite a los usuarios avanzados incrustar código HTML si fuese necesario.


🎨 2. Marcador Visual (Sección HOTSPOT)

  • Label: El texto que flotará sobre el botón (ej: "Leer historia del Aula" o "Normas del taller").
  • Icon: Permite elegir un icono representativo (suele usarse el símbolo de un libro, un periódico o una letra "i" de información).
  • Size (px) y Password: Al igual que los anteriores, regulan el tamaño del marcador en el tour y permiten bloquear la lectura del artículo con una clave.

Otros hotspot

🎥 Video Hotspot: Permite integrar un reproductor de vídeo (subido o desde YouTube/Vimeo). Ideal para que al pulsar sobre una pantalla o pizarra real del centro, se reproduzca un vídeo de presentación de la asignatura.


🔗 URL Hotspot: Funciona como un enlace directo. Al hacer clic sobre él, abre automáticamente una página web externa en una pestaña nueva (perfecto para redirigir a la web oficial del instituto, un blog de clase o un formulario de evaluación).


🎵 Audio Hotspot: Inserta una pista de sonido o locución de voz en un punto concreto del espacio. Sirve tanto para añadir efectos ambientales como para simular que un profesor o alumno te explica el aula de Viva Voz.
